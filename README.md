## Install this extension

1. Download this code
2. Click on Code(green button) and download as zip
3. Open chrome://extensions
4. Enable Developer mode (on left hand side)
5. Click on Load package button and select the folder you just downloaded!

## Features
1. SHIFT + Click Inspired by Arc's(browser) PEEK feature.
2. Consistent design mostly where it matters!!
